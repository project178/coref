# Projet "Résolution de coréférence"

Ce projet est réalisé dans le cadre du cours **Sémantique computationnelle** à l'université **Paris 3**.

Il fallait essayer un algorithme afin de trouver des chaines de coréférence dans le texte.

Un corpus Democrat a été utilisé.
Deux algorithmes ont été essayés: CRFs et Seq2Seq.
La métrique B3 a été implémenté.
